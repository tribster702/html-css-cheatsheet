HTML and CSS Reference Sheet

HTML Elements

Tag Name Description
<h1> Heading Defines first level of HTML headings
<h6> Heading Defines last level of HTML headings
<img> Image Defines an image
<div> Division Defines a division or section in an HTML document
<p> Paragraph Defines a paragraph
<link> Link Used to link to external style sheet

CSS Properties

Tag Name Description
color Font Color Sets the color of a text
background-color Background Color Sets the background color of an element
font-family Font Style Sets the font style for an element
margin Margin Sets all the margin properties in one declaration
padding Padding A shorthand property for all the padding properties
position Position Specifies the type of positioning method used for an element (static, relative, absolute or fixed)
